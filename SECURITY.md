# Security Policy

## Scope & intended use

Kitchen Micro-Segmenter generates firewall configuration. It does **not** apply anything and does
not connect to your devices — you supply an inventory file and review the output. Use it only on
networks you own or are authorized to configure.

## Reporting a vulnerability

Report security issues **in this tool** privately, not via public issues:

- GitHub **Report a vulnerability** (Security → Advisories) on this repo, or
- email the maintainer listed on the repo profile.

Include a description, reproduction, and impact. Target: **5 business days** to acknowledge, with a
coordinated disclosure timeline agreed before any public write-up. Mirrors the posture of the
Technika11y Incident Disclosure Toolkit.
